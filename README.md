## Sobre o projeto

O **elfood** é um site para compra e entrega de comida baseado no iFood.

---

## Tecnologias utilizadas

Esse projeto foi desenvolvido utilizando as seguintes tecnologias abaixo:

- Ruby on Rails (API)
- Next.js
- TypeScript
